# DockerDrupal

### Description

Skeleton for [Drupal](https://www.drupal.org/) development environment with Docker.

### Use

Clone repository and put Drupal app inside /code. Just (Docker it's a must have of course) **docker-compose up -d** and start coding!

### Comes with

1. Nginx 1.10
2. PHP 5.6
3. MySQL 5.6
